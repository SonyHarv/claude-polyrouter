# Enable Continuous Learning

## Instructions

1. Read `~/.claude/polyrouter/config.json` (create if it doesn't exist)
2. Set `learning.enabled` to `true`
3. Write the updated config back
4. Confirm to the user: "Continuous learning enabled. Polyrouter will suggest extracting insights every 10 queries."
